# Contributing

Thanks for helping build the Scratch Bugs & Tricks Hub.

## Add a New Entry

1. Create a new Markdown file in `entries/`.
2. Use lowercase words separated by hyphens, such as `sprite-layering-glitch.md`.
3. Follow the template from `README.md`.
4. Add your entry to the tracker table in `README.md`.
5. Keep instructions beginner-friendly and Scratch-specific.

## Entry Style

- Use "The Issue" for the bug, limit, or confusing behavior.
- Use "The Trick" for the workaround.
- Use "Best For" to help readers decide when to use it.
- Use "Watch Out For" for tradeoffs and edge cases.

## Contributor Submissions

- It adds one focused bug or trick.
- It describes the Scratch blocks or logic in plain English.
- It avoids clickbait claims like "unlimited clones" unless the workaround honestly explains the performance cost.
- It keeps the tone helpful and welcoming.

## Moderator Updates

Moderators can commit approved fixes directly if the repository branch rules allow moderator bypass. If GitHub rejects a push with `GH006: Protected branch update failed`, the branch protection rule still requires pull requests for that branch.
